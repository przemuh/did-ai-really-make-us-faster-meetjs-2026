We needed to understand not just WHO was using AI, but HOW WELL. So we built a weighted effectiveness score. Agent requests are worth the most — that's agentic, autonomous AI work. Applied suggestions show deliberate acceptance, not just browsing. Tabs are the weakest signal — shown but not always used. -- click -- When we ran this across the org: only 2% were Super Power Users. 10% total were using AI to its full potential. The other 90% had adoption — but not depth.

TRIANGULATION  —  Did engineers who felt faster actually ship faster?
We had the formula. But generating this score required data that wasn't available in any single dashboard. Vendor analytics were optimized for showcasing usage — not ROI. So we built our own pipeline. -- click -- Usage: raw Cursor logs. We could see agent requests versus tab completions — intentional use versus passive. -- click -- Impact: GitHub data. Cycle time, revert rates — the objective record of what actually shipped. -- click -- Trust: surveys. Per-PR impact scores, quarterly sentiment. The human signal. -- click -- This triangulation was critical. It let us validate the gap — checking if engineers who felt faster were actually shipping faster. Spoiler: it wasn't always the same people.
